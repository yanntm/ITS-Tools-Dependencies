This page hosts content regularly rebuilt using the continuous integration service Travis. This page hosts these dependencies for ITS-Tools :

  • Spot (see Spot homepage)
  • LTSmin (see LTSmin homepage)
These artifacts are not necessarily very portable, they're used in our build system and are specific to a docker hosted on travis-ci.

Both of these software are free open source packages. Please see the sources of this project to see exactly how we build them on Travis and AppVeyor.

Download the latest Spot binaries for linux x64 : Spot for Linux latest.
Note that this executable is statically linked and should work without any further installation.
